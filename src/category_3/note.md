# 笔记软件

### 文字型笔记软件

1. [Obsidian](https://obsidian.md)

   作为尝试遍了市面上各种主流软件的软件工程师，我强烈向您推荐使用 Obsidian。只要您有 Markdown 的基础，Obsidian 开箱即可满足您的笔记需求。如果您有更复杂的需求，别着急更换软件，去看看它丰富的插件生态系统有没有一款能正好解决您的问题。它的所有笔记都是离线保存在您本地的 Markdown 文件，因此信息安全可以得到充分保障。但正因为这种组织方式，您如果需要在线编辑，可能会绕一些弯路。

2. [Notion](https://notion.so) 与 [Wolai 我来](https://wolai.com)
   
   作者是两款软件的付费用户，但是非常负责任地告诉您，您使用免费版已经足够了！关于两款软件的相似性、孰优孰劣已经有很多网友在讨论了，您可以根据自己的需要选择合适的一款使用。绝大部分的功能两者都是相同的，但 Notion 的跨软件生态系统要强于 Wolai，而 Wolai 的本地化和便捷性要优于 Notion。这也是笔者两者都在使用，而且存储者不同类型的数据的原因了。
